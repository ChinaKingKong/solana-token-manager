# Solana Token Manager

一个基于Vue3、TypeScript和Solana区块链的代币管理系统，用于可视化管理和操作Solana合约。

## 功能特性

- 代币列表：查看您的Solana代币，包括SOL和SPL代币
- 创建代币：创建新的Solana SPL代币
- 创建关联账户：为代币创建关联账户
- IPFS上传：上传文件和JSON元数据到IPFS
- 设置Metadata：为代币设置和更新元数据
- 铸造代币：增发代币到指定账户
- 销毁代币：销毁持有的代币
- 冻结管理：冻结和解冻代币账户

## 技术栈

- Vue 3 (使用Composition API和setup语法糖)
- TypeScript
- Ant Design Vue
- Solana Web3.js
- Solana SPL Token
- IPFS (模拟)

## 开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
